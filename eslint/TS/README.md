# Install the Linter for TS

Copy and paste the following command to install the dependencies, then paste **.eslintrc.ts.cjs** into your repo.
Don't forget to also paste ts.config.json into your repo.

```
npm i -D typescript @typescript-eslint/parser @typescript-eslint/eslint-plugin eslint-config-airbnb-typescript eslint-import-resolver-typescript

```
